# Richer metadata fields

Richer/more structured fields than what exists today: composer/year/lyricist for Songs, free-text `name` for Recordings.

One Recording field is now decided rather than deferred: `kind` (`released` / `video_capture`, future `user_upload`) — see [domain-model.md](../domain-model.md) and [streaming-platform-links.md](streaming-platform-links.md). The free-text `name` / artist-attribution question remains open.
